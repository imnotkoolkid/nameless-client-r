# Nameless Client(r)


## Build Instructions

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone this repository (or download the source code):
   ```bash
   git clone https://github.com/imnotkoolkid/nameless-client-r/
   ```
2. Open your terminal in the project folder and install dependencies:
   ```bash
   npm install
   ```

### Building the App
To build the client for your current operating system, run:
```bash
npm run build
```
The compiled executable will be located in the `dist` folder.
